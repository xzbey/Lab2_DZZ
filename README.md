# Документация для программы "Расчет элементов плановой аэрофотосъемки"


## Функциональность программы

Программа предназначена для автоматизации расчетов элементов плановой аэрофотосъемки на основе заданных исходных данных.
Расчеты выполняются по формулам, описанным в *лабораторной работе №2* по предмету "*Технологии дистанционного зондирования Земли и фотограмметрии 3 семестр*".

**Основные возможности**
- **Ввод исходных параметров**
	- скорость самолета (W)
	- размер кадра (l<sub>x</sub>, l<sub>y</sub>)
	- фокусное расстояние (f)
	- масштаб аэрофотосъемки (m)
	- масштаб фотоплана (M)
	- нормативные перекрытия (продольное - p<sub>x</sub> / поперечное - p<sub>y</sub>)
	- длина участка (L<sub>x</sub>, L<sub>y</sub>)
	- высота рельефа (максимальная - Z<sub>max</sub> / минимальная - Z<sub>min</sub>)
- **Автоматический расчет 15 ключевых параметров (1-12 шаги)**
	- 1 шаг - площадь участка (S)
	- 2 шаг - высота полета (H)
	- 3 шаг - перекрытия (продольное - P<sub>x</sub> / поперечное - P<sub>y</sub>)
	- 4 шаг - размер рабочей области (по x - b<sub>x</sub> / по y - b<sub>y</sub>)
	- 5 шаг - базис фотографирования (B<sub>x</sub>)
	- 6 шаг - количество маршрутов (K)
	- 7 шаг - снимков в маршруте (n)
	- 8 шаг - общее количество снимков (N)
	- 9 шаг - длина пути самолета (L<sub>S</sub>)
	- 10 шаг - время съемки (T<sub>S</sub>)
	- 11 шаг - максимальная выдержка (t<sub>max</sub>)
	- 12 шаг - интервал съемки (r)
- **Отображение результатов в понятном формате с единицами измерения**
- **Дробный результат округляется и выводится с 3-мя знаками после запятой**
- **Очистка результатов при необходимости или изменении входных данных**

<br>Программа сокращает время ручных расчетов, минимизирует ошибки и позволяет быстро экспериметрировать с различными данными.

## Использование программы
- **Запуск программы**
	- Скачайте файлы `main.html`, `styles.css` и `scripts.js`
	- Откройте файл `main.html` в любом браузере
- **Ввод данных**
	- В разделе **`Исходные данные`** заполните поля. Значения по умолчанию соответствуют примеру из лабортаторной работы:
		- W = 320 км/ч
		- l<sub>x</sub> = 18 см
		- l<sub>y</sub> = 18 см
		- f = 0.22 м
		- m = 13200
		- M = 10000
		- p<sub>x</sub> = 60%
		- p<sub>y</sub> = 30%
		- L<sub>x</sub> = 25 км (+12км запаса)
		- L<sub>y</sub> = 30 км (+12км запаса)
		- Z<sub>max</sub> = 212 м
		- Z<sub>min</sub> = 182 м
	- Измените значения, если нужно (программа автоматически очистит результаты при изменении)
- **Расчет**
	- Нажмите кнопку **`Посчитать`**
	- Результаты отобразятся в разделе **`Расчетные результаты`** с указанием шага и единиц измерения
- **Очистка**
	- Нажмите кнопку **`Очистить результаты`**, чтобы очистить результаты
- **Примечания**
	- Программа может работать оффлайн (без интернета)
	- Результаты округляются до 3-х знаков после запятой
	- Если ввод не корректен (не число), программа выдает NaN в результате

## Реализация
- **Языки программирования**
	- **`HTML5`** - для структуры интерфейса
	- **`CSS3`** - для стилей
	- **`JavaScript`** - для логики расчетов
- **Библиотеки**
	- Не используются внешние библиотеки/фреймворки.

## Ключевые фрагменты кода с расшифровкой

### scripts.js
- **Функция получения значения из input - `get(id)`**
	Получает значения из HTML-элемента по id, преобразует в число. Используется для чтения всех входных данных, сделана для избежания повторений
- **Функиця округления числа - `fix(num)`**
	Округляет число до 3-х знаков после запятой и возвращает как число. Используется для всех расчетов, чтобы результаты были точными (*Пример: 2904.000 -> 2904*)
- **Основная функция расчета - `result()`**
	Собирает все входные данные. Выполняет расчеты шаг за шагом по формулам из лабораторной. В конце обновляет текст в HTML-элементах результатов. Также включается преобразования единиц (*Пример: шаг 11-12 - W переводится в м/с*)
- **Функция очистки `clearResults()`**
	Сбрасывает все поля результатов в исходное состояние "Не вычислено". Вызывается по кнопке или при изменении input.
- **Привязка событий `document.getElementById(element).addEventListener(signal, function)`**
	Добавляет обработчики кликов на кнопки

### main.html
Структура интерфейса. Определяет поля ввода, кнопки, лейблы с id для доступа из JS и CSS.

### styles.css
Определяет стили для кнопок, полей ввода, лейблов, текстов и фона.
